Giới Thiệu

Đây là một ứng dụng web demo được xây dựng bằng Flask framework của Python.

Về Flask

Flask là một micro web framework được viết bằng Python. Nó được phân loại là microframework vì nó không yêu cầu các công cụ hoặc thư viện cụ thể.

Công Nghệ Sử Dụng

  • Backend: Python Flask
  • Frontend: HTML, CSS, JavaScript
  • Template Engine: Jinja2
  • Styling: Custom CSS

Tính Năng

  • Routing và URL handling
  • Template inheritance
  • Form handling
  • Flash messages
  • Error handling
  • JSON API endpoints
